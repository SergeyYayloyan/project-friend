Заметка для друга по Git

cd .. - перейти на уровень выше
cd projects - перейти в папку projects
mkdir (название) - создать папку (название)
rmdir (название) - удалить папку (название)
rm -r (название) - удалить папку (название) и все содержимое
touch hello.txt - создать файл hello.txt в текущей папке
clear - очистить терминал
git init - инициализировать
git status - текущее состояние репозитория
